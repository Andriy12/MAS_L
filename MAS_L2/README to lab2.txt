Оскільки суттєвим недоліком(на мою думку) агента BookBuyerAgent є те, що книга яку він має купити задається статично при його створенні, 
я реалізував можливість динамічної зміни книги для покупки підчас роботи даного агента. Кожні 20 секунд агент перевіряє, чи в нього є назва книги
для покупки, якщо є то виконуються основні алгоритмми що реалізують процеси покупки, якщо ні то агент залишається бездіяльним. Для реалізації даної 
задачі я написав клас BookBuyerGui де задається ім*я книги для покупки.
